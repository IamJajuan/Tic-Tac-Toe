## General info
It is a Tic-Tac-Toe game with multiplayer functionality and a simple AI for single player mode


## Technologies
Project is created with:
* react version: 17.0.2
* express version: 4.16.1
* socket.io version: 4.1.3


## Setup front-end
```
$ cd front-end
$ npm install
$ npm start
```

##  Setup back-end
```
$ cd back-end
$ npm install
$ npm start
```
